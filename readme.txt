== Description ==

Das Plugin gibt Google Trends Graphen per Shortcode aus, zudem kannst Du die Top-Suchanfragen bei Google in einem Widget ausgeben.

Beispiel:

[trends h="500" w="500" q="cats,dogs,+cute+dogs,+cute+cats,+cats+and+dogs" geo="de"]
